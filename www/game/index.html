42 Don't Panic

  • Digite uma equação cujo o resultado seja 42
  • Digite apenas numeros e operadores
  • Se a letra está no lugar certo: [C]
  • Se a letra está no lugar errado: [T]
  • Se a letra não existe na palavra: [X]
  • Se a quantidade de caracteres for diferente de 6 "TAMANHO ERRADO"
  • Para enviar apertar: [ENTER]
  • Bom Jogo!
